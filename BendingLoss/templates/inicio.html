Our Application

This web application is designed to measure the weight of objects in real-time using a fiber optic sensor. By connecting the USB the app can read the serial port from the browser, the application captures and visualizes measurement data interactively and accurately.

How It Works

Calibration: Calibration is the essential first step to ensure measurement accuracy. In the calibration tab, the system starts capturing data from the sensor to determine the stable voltage when the scale is unloaded (0 kg). This reference value is crucial as all subsequent measurements will use this stable voltage as a baseline to calculate weight changes.

Real-Time Weighing: Once calibration is completed, users can proceed to the weighing tab. During this phase, the application uses the previously determined stable voltage to calculate the decibels (dB) corresponding to the weight of the object placed on the scale. Results are graphed in real-time, providing an immediate visualization of weight variations.

History: To facilitate analysis and review, the application allows users to save the graphs generated during measurements. Saved graphs can be reviewed in the history tab, offering a comprehensive view of past measurements and allowing comparisons between different weighing sessions.

You can also use this app to display real time data of any microcontroler programmed to send data from the serial port at a baudrate of 9600
